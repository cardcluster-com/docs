## Neue Karten, Reprints und Pre-Releases
Jede Karte eines Sets hat eine Nummer auf der linken Seite der Liste. Die Kartennummer von neuen Karten ist blau, die von Reprints grau und die von Pre-Releases gelb. Pre-Releases sind Prints von Karten, die ihren offiziellen ersten Print zu einem späteren Zeitpunkt in einem Core Booster oder Hidden Arsenal Set haben.

## Kartentexte
Bestimmte Teile des Kartentexts werden automatisch hervorgehoben. Die Hervorhebung unterstützt die Struktur des Problem-Solving-Kartentexts
und erhöht die Lesbarkeit insbesondere bei langen Effekten.

| Bezeichnung | Farbe / Formatierung |
| -------- | ------- |
| Beschwörungsmaterialien von Extra Deck Monstern  | grauer, fetter Text |
| Aktivierungsbedingung | blau |
| Kosten | rot |
| Effekt | *keine* |
