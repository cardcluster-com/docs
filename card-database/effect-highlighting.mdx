## New cards, reprints and pre-releases
Each card in a set has a number on the left-hand side of the list. The card number of new cards is blue, that of reprints is gray and that of pre-releases is yellow. Pre-releases are prints of cards that have their official first print at a later date in a Core Booster or Hidden Arsenal Set.

## Card texts
Certain parts of the card text are automatically highlighted. The highlighting supports the structure of the Problem-Solving Card Text and increases readability, especially for long effects.

| Description | Color / Format |
| -------- | ------- |
| Summon materials of Extra Deck monsters  | gray, bold text |
| Activation requirement | blue |
| Costs | red |
| Effect | *none* |
